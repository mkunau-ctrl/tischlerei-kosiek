# Tischlerei Kosiek — Demo-Website

Konzept-Demo für die Tischlerei Carsten Kosiek aus Lemgo. Statische 5-seitige Website im warm-rustikalen Handwerksdesign.

## Live

Sobald GitHub Pages aktiviert ist, läuft die Site unter `https://<github-username>.github.io/tischlerei-kosiek/`.

## Stack

Reines HTML5 + CSS3 + Vanilla JS. Keine Frameworks, kein Build-Tool. Schriften via Google Fonts (Playfair Display, Lora, Cormorant Garamond). Karte via OpenStreetMap.

## Seiten

- `index.html` — Startseite mit Hero, Leistungs-Teaser und Testimonial
- `leistungen.html` — Übersicht aller acht Leistungsbereiche
- `ueber-uns.html` — Vorstellung des Betriebs und der Werkstatt
- `galerie.html` — Bildergalerie mit Filter und Lightbox
- `kontakt.html` — Kontaktdaten, Formular und Karte

## Hinweis

Diese Website ist ein unverbindlicher Entwurf zur Demonstration und kein offizielles Angebot der Tischlerei Kosiek. Alle eingesetzten Bilder sind Platzhalter-Fotos von Unsplash.
